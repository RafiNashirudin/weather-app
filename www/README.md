#Install Cordova CLI

install cordova
```bash
npm install -g cordova
```

Buat Project Baru:
```bash
cordova create myApp com.example.myApp MyApp
cd myApp
```

Tambahkan Platform Android
```bash
cordova platform add android
```

Uji Aplikasi di browser (Opsional):
```bash
cordova serve
```

Bangun aplikasi android
```bash
cordova build android
```
